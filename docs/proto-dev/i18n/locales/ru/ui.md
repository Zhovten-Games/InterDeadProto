# Russian UI Strings

Русская локализация для экранов управления, AI-лоадера, лаунчера чата и регистрации профиля. Здесь же собраны ключи диалогов духов (`guide.*`, `guest1.*`), подсказки для импорта/экспорта профиля и тексты финальных реакций.[^1]

Добавлены строки подтверждения переключения духа (`ghost_switch.*`) и AI-индикаторов, чтобы панель управления и оверлей AI могли показывать корректные подсказки и статусы.[^2]
