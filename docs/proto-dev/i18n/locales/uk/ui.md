# Ukrainian UI Strings

Українська локалізація для UI-кнопок, AI-лоадера, лаунчера чату та реєстрації профілю. Тут же зібрані ключі діалогів духів (`guide.*`, `guest1.*`), описи імпорту/експорту профілю та тексти фінальних реакцій.[^1]

Передбачені строки для підтвердження перемикання духа (`ghost_switch.*`) і статуси AI, щоб панелі керування та оверлей AI показували правильні підказки та стани.[^2]

[^2]: AI/launcher статуси та підтвердження перемикання духа [src/i18n/locales/uk/ui.json#L21-L105](../../../../src/i18n/locales/uk/ui.json#L21-L105)
